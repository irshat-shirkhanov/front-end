
1) git init - подключаем созданный проект к git.


2) Создаем файл ".gitignore" и в него происываем:
  **/*.sass - эта команда позволит игнорировать все файлы с разрешением sass в нашем проекте;
  /bin/ - эта команда игнорирует папку bin
  /admin/ - эта команда игнорирует папку admin
  /config/ - эта команда игнорирует папку config

3)
3.1 git add . - добавляем все файлы в локальное хранилище;
3.2 git commit -m 'added all files' - коммитим добавленные файлы;
3.3 git branch newbrach - создаем ветку newbrach;
3.4 git checkout newbranch - переходим на новую ветку;
4.5 Здесь создаем папку blog с файлами index.js, index.html;
4.6 git checkout master - переходим на основную ветку проекта;
